# DIFFERENTIAL DELTAN // Portfolio

A high-fidelity digital showcase for **Navdeep Singh**, operating at the intersection of raw street culture and deliberate visual chaos. This portfolio bridges the gap between vibrant grandiosity and psychological weight.

> "I don't make ART. I make statements."

## The Aesthetic
- **Visual Noise / Raw Energy / Intentional Chaos**
- High-contrast, brutalist typography.
- Interactive custom cursor logic with lerped follower states.
- Dark-themed UI with accent-driven navigation (Vault, Architect, Comms).

## Technical Implementation
- **Frontend:** Pure HTML5, CSS3, and JavaScript (ES6+).
- **Smooth Interaction:** Custom-built `lerp` and `clamp` utility functions for a cinematic cursor lag effect.
- **Design System:** Assets crafted in Photoshop, focusing on poster art and album cover design for the modern trap/hip-hop scene.

## The Vault (Selected Works)
- **Leonardo DiCaprio:** High-noise, minimalist print style.
- **Young Thug:** Multi-layered, textured composition.
- **Drake Graham:** Typographic-heavy editorial layout.
- **Fyodor Dostoevsky:** Philosophical textured portrait.
- **Nihilist Penguin:** Atmospheric, atmospheric scenery design.
